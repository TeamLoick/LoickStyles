# Tutorial Intro

Let's discover **LoickStyles**.

## Getting Started

Get started by **importing what do you need**.

### Global Import

This is required if you want to use loick styles

```html
<link
  rel="stylesheet"
  href="https://teamloick.github.io/LoickStyles-CDN/css/global.css"
  async
  defer
/>
```
